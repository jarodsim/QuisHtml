Quiz sobre HTML

1 - a tag <!DOCTYPE html> diz para o navegador que o tipo de documento que ele está abrindo é um arquivo HTML. Essa tag NÃO é excluiva para o HTML, por exemplo. Também é usada para dizer para o navegador que ele está abrindo um arquivo .xml... EX: <!DOCTYPE xml>.

2 - a tag <i> deixa o texto em itálico. EX: <i> Sou um texto em itálico </i>

3 - a tag <u> deixa seu texto sublinahdo, ou seja, com uma linha embaixo dele. Como vem do inglês "Underline" que traduzindo para o português fica: "sublinhado", é justamente por isso que a tag é <u>, "u" de "underline".

4 - a tag <link> precisa de atributos e valores nesses atributos. Ela não é só usada para indicar css externo, mas também para outros tipos de arquivos e necessidades.

  • - O atributo "rel="stylesheet"" serve para você dizer para o navegador que é uma folha de estilo que você quer "pegar" para seu site.
  • - O atributo " type="text/css"" diz que o tipo de arquivo que você está chamando pode ser um .txt/.css.
  • - O atributo "href="estilo.css"" nada mais que é onde você coloca o endereço de seu css externo.

5 - O HTML é uma linguagem de marcação de texto.
